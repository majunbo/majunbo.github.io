用hexo框架搭建静态博客

2020-04-05

【Hexo 静态博客框架』是一个纯静态的博客网站框架，也就是一款基于 Node.js 的静态博客框架，依赖少、易于安装使用，可以方便的生成静态网页托管在服务器上，并且Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

这套软件官网：https://hexo.io/zh-cn/index.html

Hexo博客的开发与部署流程
接下来简单阐述一下一个之前从未接触过Hexo的折腾党是如何搭建并部署的：

1)安装hexo软件所有操作命令

# 所有操作步骤
npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server

4)安装自定义主题

hexo主题页面：https://hexo.io/themes/

# 安装landscape-plus主题
[root@web blog]# git clone https://github.com/xiangming/landscape-plus.git themes/landscape-plus
[root@web ~]# vim /mnt/blog/_config.yml 
change this:
theme: landscape
to this:
theme: landscape-plu

5)部署静态页面到nginx

软件安装完成了，静态页面也生成了，肯定要使用世界上最牛的nginx来跑咱的博客系统了。

# 编辑nginx配置文件，指向我们新生成的目录
[root@web ~]# vim /usr/local/nginx/conf/nginx.conf
change this:
        location / {
            root   root;
            index  index.html index.htm;
        }
to this:
        location / {
            root   /mnt/blog/public;
            index  index.html index.htm;
        }
# 也可以根据需要适当修改其他配置参数。

# 重新启动nginx
[root@web ~]# systemctl restart nginx.service
